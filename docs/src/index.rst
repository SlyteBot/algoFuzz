Welcome to AlgoFuzz's documentation!
===================================

**AlgoFuzz** (/al'gow'fuhz/) is a Python library for popular fuzzy c-means clustering algorithms from the literature.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
